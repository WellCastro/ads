# Executando a aplicação localmente:
```
cd ads
```
```
docker-compose build
```
```
docker-compose up
```
```
docker-compose python manage.py runserver migrate
```
```
docker-compose up -d
```

# Tests
```
cd tests
```
```
pytest tests.py
```
